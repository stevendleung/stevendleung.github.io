*Conference talk at [CAMLIS 2025](https://www.camlis.org) (Conference on Applied Machine Learning for Information Security).*

This presentation examines Risk-Based Authentication (RBA) effectiveness in production two-factor authentication (2FA) systems against real-world attacks. The study evaluates how heuristic and anomaly detection methods strengthen security without compromising user experience.
